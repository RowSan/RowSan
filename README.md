## Featured Project — StoneCat

StoneCat is a warehouse intelligence system designed to improve 
inventory visibility and reduce manual dependency on ERP workflows.
[Read the full 1-page StoneCat Case Study here](./docs/StoneCat-Case-Study.md)

### Core Capabilities

- Data ingestion pipeline from ERP (Fishbowl CSV → Supabase)
- Structured inventory model supporting multi-warehouse visibility
- Business logic for stock state interpretation and pricing validation
- Batch-level data linking (dimensions, location, media)
- Read-optimised mobile interface for real-time access

### System Design

- Decoupled architecture (ingestion → storage → presentation)
- Controlled data flow with validation rules
- Designed for performance on constrained mobile environments
